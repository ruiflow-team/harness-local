Safety defaults

This beta is intentionally conservative. It is not a cloud SaaS and does not promise automatic code changes.

✓

127.0.0.1 first

Default local bind keeps the web UI on your machine unless you explicitly change it.

✓

No default upload

Projects are not uploaded to a cloud service by default.

✓

No credential read

Credential files and plaintext API keys are outside the beta default flow.
