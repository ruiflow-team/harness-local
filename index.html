Harness Local v0.24 Beta

Local-first AI Agent execution workbench. 在你的电脑上，把 AI 从聊天变成可验证的任务流。

Download ZIP Install Guide Configuration

Quick Start

unzip harness-local-v0.24-beta.zip
cd harness-local-v0.24-beta
./harness serve

Open http://127.0.0.1:8788

Verify

SHA256:

681e070a7755140c2639833b1841b11933dd4b35e6b094785677fd1a6cfa37bc

shasum -a 256 harness-local-v0.24-beta.zip

Configuration

HARNESS_PORT=8789 ./harness serve
HARNESS_DATA_DIR=/tmp/harness-local-data ./harness serve
./harness smoke-empty

Safety Defaults

  • Default bind: 127.0.0.1
  • No default cloud upload
  • No credential file read
  • No destructive command execution
  • Optional plugins disabled by default
